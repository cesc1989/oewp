Cargar recurso via enlace

Así como en facebook, copias un enlace de un vídeo de YouTube y lo pegas en el campo para actualizar estado, y obtienes una vista miniatura del recurso, es la característica que ofrece esta práctica a continuación.

Notas

Solo funciona con vídeos de YouTube, Vimeo, Metacafe, Google Video y Dailymotion.

Si el enlace tiene el https, elimina la s al final para que funcione el código.
